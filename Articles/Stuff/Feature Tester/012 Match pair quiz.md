# Match pair quiz

How to create a match pair quiz.

```html
<Quiz>
{
  "Type": "MatchPair",
  "Title": "Match the Java Concepts",
  "Pairs": [
    {
      "Prompt": "int",
      "Answer": "A 32-bit signed integer"
    },
    {
      "Prompt": "boolean",
      "Answer": "Represents true or false"
    },
    {
      "Prompt": "char",
      "Answer": "A single 16-bit Unicode character"
    }
  ]
}
</Quiz>
```
