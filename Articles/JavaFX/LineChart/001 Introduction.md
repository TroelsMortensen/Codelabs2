# Introduction to JavaFX LineChart

This learning path introduces `LineChart` in JavaFX using the space exploration domain from Session 20.

The focus is practical: how to structure chart data, how to render it cleanly, and how to split responsibilities between ViewModel and Controller in an MVVM-style JavaFX application.

## Prerequisites

You should already be comfortable with:

- basic JavaFX controls and event handling
- FXML and controllers
- Java classes and collections

## Goal

At the end of this path, you should be able to:

1. build and configure a JavaFX `LineChart<Number, Number>`
2. map domain data to chart points
3. keep chart data in a ViewModel and keep the controller thin
4. update chart series safely when data comes from background work

## What this path covers

1. Building a basic line chart
2. Space exploration case: planet distance trends
3. MVVM data ownership (ViewModel vs Controller)
4. Dynamic updates and JavaFX thread rules
5. Exercise: mission telemetry trend
