# Introduction to Adapter Assignment

In this assignment you must adjust your system to use two provided classes. You will get the code for these classes on later pages.

You will change how your system logs information. And you will change how your system shows notifications to the user.

## Deliverables

* Updated logging functionality to use the provided class
* Updated notification to use the provided class
