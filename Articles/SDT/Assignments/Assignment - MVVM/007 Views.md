# The Views

I think you are ready to start creating the views.

## Main view

If you are going for a Single View Application, you may start with the main view.

And, if you are _not_ going for a Single View Application, it's still probably a good idea to have some kind of main view.

## Other views

It is up to you how you want to structure your views, and how many features a view covers.

Maybe you have a view just for the stock price chart.\
Maybe that view also has a button to buy a stock.\
Or maybe there is a different view to buy stocks.\
Maybe buying and selling is one view.\
You probably need a portfolio view, to see the stocks you own and their quantities, as well as your balance.\
Maybe you can sell stocks from the portfolio view.

It is all up to you.

As long as you follow the requirements on page 1.
