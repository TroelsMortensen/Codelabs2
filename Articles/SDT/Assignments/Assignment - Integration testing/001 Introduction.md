# Assignment - Integration testing

In this assignment you will write automated integration tests parts of your application. We will focus on one or two use cases, though you may choose to do more.

## Deliverables

- Integration tests for buying stock.
- Integration tests for Selling stock.

**Optional:**

Feel free to add integration tests for other uses cases or features:
- Viewing portfolio data.
- Viewing transaction history.
- Viewing stock price history.
- etc.
