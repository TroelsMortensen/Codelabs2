# Passing arguments to a view

Sometimes you want to open a view, and have that view show some specific data right away. For example, you may have a view that shows a list of items, and when the user clicks on an item, you want to open a detail view for that item. This second view needs to know which item the user clicked on, so that it can show the correct data.
