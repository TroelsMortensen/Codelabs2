# First app, with fxml and controller

Watch the following video to see how to create a simple app, with fxml and controller.

This app will be
- A TextField for inputting text
- A Label to display the text
- A button to transfer text from the TextField to the Label
- A button to clear the TextField and the Label
- And some event handling, so that the button is disabled when the TextField is empty
